# Eat ABQ Website

Marketing landing page for [Eat ABQ](https://github.com/connor5165/Eat-ABQ), an Android app that helps you decide where to eat in Albuquerque, NM by spinning a food-category wheel.

Live at [eatabq.app](https://eatabq.app), served via GitHub Pages (see `CNAME`).

## Contents

```
index.html    Landing page (hero, features, download link)
privacy.html  Privacy policy
terms.html    Terms of use
CNAME         GitHub Pages custom domain config
```

Plain static HTML/CSS — no build step, no dependencies. Edit the HTML files directly and push to deploy.
